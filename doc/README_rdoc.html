clipboard

Access the clipboard and do not care if the OS is linux, mac or windows.

Usage

You have Clipboard.write [copy],

Clipboard.read [paste] and

Clipboard.clear

Have fun ;)

Requirements

  • Linux: xclip (you can install it on ubuntu with sudo apt-get install xclip)

  • Windows: win32-clipboard gem

  • Zucker gem
